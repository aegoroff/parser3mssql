Для сборки необходимо:

1. Visual Studio 2012

2. Переменная окружения PARSER3PROJECT указывающая на местоположение каталога 
с исходниками парсера. Согласно правилам сборки парсера, это каталог parser3project 
находящийся в корне любого диска. Например если этот каталог находится на диске D, 
значение переменной должно быть D:\parser3project

3. Переменная окружения SQL_SDK_ROOT указывающая на местоположение MS SQL SDK 
(устанавливается после установки Native клиента), 
обычно это C:\Program Files\Microsoft SQL Server\90\SDK
